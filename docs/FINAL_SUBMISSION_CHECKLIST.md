# Final Submission Checklist

## Code and Repository
- [ ] GitHub repository is public
- [ ] Source code pushed to GitHub
- [ ] No secrets committed (`.env` not pushed)
- [ ] README is updated with setup, run, and deployment steps

## Project Execution
- [ ] MongoDB connection works (`MONGO_URI` configured)
- [ ] Backend runs successfully (`npm run dev --prefix backend`)
- [ ] Frontend runs successfully (`npm run dev --prefix frontend`)
- [ ] Combined run works (`npm run dev` from root)
- [ ] Demo seed works (`npm run seed` from root)

## Functional Validation
- [ ] User/Admin registration and login works
- [ ] Admin can create/update/delete bunks
- [ ] Admin can create/update/delete slots
- [ ] User can search bunks by state/district (and optional city/area)
- [ ] User can view slot vacancy and book slot
- [ ] User can cancel active booking
- [ ] Admin can update booking status
- [ ] Admin booking table supports search (`q`) and status/date filters
- [ ] Admin booking table supports sorting and pagination
- [ ] Admin summary report and CSV export work correctly

- [ ] Detailed report completed (`docs/PROJECT_REPORT.md`)
